AI Wellness SaaS Platform

Wellness360

Transforming wellness tracking with AI-powered insights and engaging user experiences

Role: UI/UX Designer ‚Äî product UI, layout system, interaction design
Tools: Figma ¬∑ Framer ¬∑ Miro ¬∑ Notion

Overview

Wellness360 is an AI-powered wellness platform designed to help users track fitness, mental health, and daily habits in one unified experience. The goal of this project was to transform a static reference into a fully structured, scalable SaaS homepage with engaging, product-focused UI.

Problem

Most wellness platforms either feel cluttered with too many features or fail to clearly communicate their value. Users often struggle to understand how AI enhances their wellness journey, leading to low engagement and poor onboarding experiences.

Goals

  • Create a modern SaaS homepage that clearly communicates product value
  • Showcase AI-driven insights in a simple, understandable way
  • Design engaging sections to improve user interaction and retention
  • Build a scalable UI system for future product expansion
  • Maintain clean, intuitive UX aligned with real-world applications

Process

  • Analyzed reference design and identified gaps in structure and usability
  • Defined content hierarchy based on SaaS best practices
  • Created layout wireframes focusing on clarity and flow
  • Designed high-fidelity UI with modern components and spacing system
  • Refined interactions and visual hierarchy for better engagement

Key Design Decisions

  • AI Insights Section: Visual cards explaining personalized recommendations in a simple format
  • Fitness Reels UI: Interactive content blocks inspired by short-form media for engagement
  • Wellness Programs: Structured sections highlighting guided plans and benefits
  • Micro-CTAs: Subtle action buttons across sections to encourage interaction
  • Visual Hierarchy: Use of gradients, cards, and spacing to guide user attention
  • Scalable Design System: Consistent components for easy future expansion

Final Design / Screens

Hero section, AI insights, reels section, wellness programs, CTA sections

Outcome

Delivered a clean, modern SaaS homepage that balances functionality and visual appeal. The design improves clarity of AI-driven features while maintaining strong engagement through interactive sections and structured layout. The scalable UI system ensures the product can grow without losing consistency.

Key Impact:

  • Improved clarity of AI features through visual storytelling
  • Increased engagement via interactive content sections
  • Created a reusable UI system for future product scaling
  • Designed with real-world SaaS usability standards
